🎓 Grade Management System

A console-based Java application to manage and analyze student academic performance efficiently.

🚀 Features
Add student details with marks
Calculate average marks automatically
Assign grades based on performance
View all student records
Generate reports (top performers & grade distribution)
Menu-driven interface for easy interaction
🛠️ Tech Stack
Language: Java
Concepts Used:
Arrays (1D & 2D)
ArrayList
Object-Oriented Programming (OOP)
Loops & Conditional Statements
Scanner (for input handling)

⚙️ How to Run
1. Open terminal

Navigate to the project folder

2. Go to source folder
cd GradeManagementSystem/src
3. Compile the code
javac *.java
4. Run the program
java GradeManagementSystem
