Grade Management System
Project Overview
The Grade Management System is a console-based Java application developed to manage student academic performance using arrays and object-oriented programming concepts.

This system allows users to store student marks for multiple subjects, calculate averages, assign grades, and generate performance reports. The application provides a menu-driven interface for easy interaction and demonstrates real-world data handling using Java.

The project was developed as part of the Developer's Arena Internship Program to strengthen understanding of control flow, arrays, and modular programming.

Features
Add student marks for multiple subjects
Store data using arrays and ArrayList
View all student records
Calculate student averages
Assign grades (A+, A, B, C, D, F)
Generate performance reports
Display top performers
Show grade distribution
Menu-driven user interface
Input handling using Scanner
Technologies Used
Java
Arrays (1D and 2D)
ArrayList
Scanner class
Object-Oriented Programming (OOP)

Setup Instructions
Follow these steps to run the project on your system:

Step 1: Open Terminal
Navigate to the project folder.

Step 2: Go to Source Folder
cd GradeManagementSystem/src

Step 3: Compile the Program
javac *.java

Step 4: Run the Application
java GradeManagementSystem
